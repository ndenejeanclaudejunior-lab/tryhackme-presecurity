## 🧠 Ce que j’ai compris
- HTTP est un protocole de communication web
- HTTPS est sécurisé (chiffrement)
- Le client est le navigateur qui demande, le serveur est la machine qui répond
- Les ports sont des "portes" d’accès aux services sur le serveur
