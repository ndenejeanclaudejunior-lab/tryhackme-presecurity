# 🛠 Tools Basics

## 📚 Ce que j’ai appris
- Nmap : scanner réseau et ports
- Netcat : envoyer/recevoir données sur un port
- Burp Suite : tester sécurité web
- Wireshark : capturer et analyser le trafic

## 🧠 Ce que j’ai compris
- Nmap permet de connaître les machines et services actifs
- Netcat est un outil polyvalent pour tester la connexion
- Burp Suite intercepte les requêtes pour détecter des vulnérabilités
- Wireshark aide à comprendre le trafic réseau en détail

## 🚀 Approfondissement
Pour mieux comprendre, j’ai testé chaque outil dans des laboratoires Pre-Security.  
Je confirmerai mes compétences lors de challenges réels ou exercices pratiques.
