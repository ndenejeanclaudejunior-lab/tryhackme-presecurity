## 🧠 Ce que j’ai compris

### 🔹 Adresse IP
Une adresse IP sert à identifier une machine sur un réseau.  
Il existe :
- des IP privées (utilisées dans un réseau local)
- des IP publiques (utilisées sur Internet)

### 🔹 DNS (Domain Name System)
Le DNS permet de traduire un nom de domaine en adresse IP.

Exemple :  
google.com → 142.xxx.xxx.xxx

L’obtention de l’adresse IP se fait en plusieurs étapes :

1. Cache (mémoire locale)
2. DNS Resolver (serveur DNS)
3. Root Server (serveur racine)
4. TLD Server (.com, .org…)
5. Authoritative Server (donne l’IP exacte)
6. Retour de l’IP à la machine
7. Connexion au serveur
