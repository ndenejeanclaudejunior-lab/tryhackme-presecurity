# 🐧 Linux Basics

## 📚 Ce que j’ai appris
- Systèmes Linux (Ubuntu, Kali)
- Commandes de base : ls, cd, pwd, cat
- Gestion des fichiers et dossiers
- Permissions et super-utilisateur

## 🧠 Ce que j’ai compris
- Le terminal est l’interface pour taper des commandes
- `ls`, `cd`, `pwd`, `cat` permettent de naviguer et lire les fichiers
- Les permissions contrôlent qui peut lire/écrire/exécuter
- `sudo` permet d’exécuter des commandes avec des droits élevés

Pour mieux comprendre Linux, j’ai suivi des tutoriels et pratique directement dans un terminal.  
Je pourrai confirmer mes connaissances en situation réelle, par exemple lors de challenges Pre-Security.
